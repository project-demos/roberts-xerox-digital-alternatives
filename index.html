What are the main components of the Digital Alternatives solution?

Digital Alternatives consists of several interconnected components. All are included when a customer buys the solution.

  • Digital Alternatives Server software
  • Digital Alternatives PC client
  • Digital Alternatives Apple iPad client

How is Digital Alternatives sold?

  • Digital Alternatives is available as a private cloud or on-premise deployment within Enterprise accounts, on a per-user, per-month basis. 
  • An implementation fee is incurred at time of install, and is estimated at 12 hours analyst time for standard installations.
  • Xerox installs the Digital Alternatives application on customer provisioned servers.
  • Each Digital Alternatives seat provides a user with 5 GB file storage allocation, and the ability to connect any number of end user client devices using the same account. The storage may be increased or decreased for any user or group of users based on preference.
    • A PC client is made available to Enterprise users via silent push
    • A PC client (.exe) download is available for secondary PC computers, such as a second laptop or home PC.
    • A free end-user Apple iPad client is available for download from the Apple app store. Note that similar to the PC client application, the Apple iPad client must successfully authenticate the user using correct Active Directory credentials for a given customer implementation before the app is functional.

Is Digital Alternatives Cloud-based?

Digital Alternatives may be deployed as a private cloud, or hosted on-premise via server at the Client site. The server is configured to provide 5 GB of storage per user account for synchronizing across devices. Additionally, Digital Alternatives provides a routing capability using Windows Azure between an end user’s device outside of the corporate LAN and the on premise server protected by network firewall. Therefore Digital Alternatives provides remote synchronization and file share capabilities, with the security and centralized management advantages of an on premise solution.

What are some customer use scenarios for Digital Alternatives?

During our Early Customer Acceptance Test (ECAT) and Customer Advisory Council (CAC) engagements, we have learned a lot about how customers perceive the solution, and what applications they envision using it for. Here are just a few:

  • A university is considering the opportunity for students to collaborate with a satellite campus on another continent.
  • Another large university is considering replacing an Enterprise File Sync and Share (EFSS) system in place todaywith Digital Alternatives, due to the rich, easy to use annotation capabilities of Digital Alternatives.
  • A car leasing service is interested in quickly completing forms using the guided form completion capabilities of Digital Alternatives.
  • A healthcare system is looking at options for Digital Alternatives to help address the thousands of forms that aren’t already converted to e-PDF documents.
  • A professional services client sees the potential for paperless meetings with Digital Alternatives’ PowerPoint viewing and annotation capabilities.
  • A shipping logistics customer is considering using Digital Alternatives for annotating and signing documents related to shipping and customs, instead of the digital signature utility currently under consideration

How does Digital Alternatives capture information and perform the analytics? Based on each individual document cycle? Over a certain period of time? Is every user action logged?

Organizational document and user activity is logged into a database on the Digital Alternatives server, without transmission of sensitive document or personal data. This forms the basis of the analytics, which is accessed via the analytics tool and dashboard. This uses the same reporting engine as Xerox’s industry-leading AssetDB, and can be combined with other usage data. As a result, Xerox can help enterprises not just track document printing and output, but also better understand and manage their migration from paper to digital over time. Customers concerned over the transmission of data for reporting purposes may have this service disengaged at any time.

Can the application be customized based on department if there are additional specific functionality needs, or, that need to be removed (i.e. perhaps a certain department doesn’t have authorization to add signatures)?

Currently it is not possible to customize the application. This is on the roadmap for future consideration however.

Are there any restrictions surrounding document file size or type that can be used or shared within the application?

Digital Alternatives can import any file type that can be converted to a PDF file. There are no restrictions on the size of file that can be stored, shared or synched with Digital Alternatives. If documents exceed maximum file sizes set by corporate email policies, they will still be made available within the application for sharing with other Digital Alternatives collaborators.

Is formal training required for end users?

Digital Alternatives is very simple and intuitive. Users should be able to master it without formal training. Users may access the video tutorials at (www.Xerox.com/DAdashboard). For software, documentation or more support, go to (www.xerox.com/support) and search “Xerox Digital Alternatives”.

How does Digital Alternatives differ from standalone signing or note-taking software?

Digital Alternatives supports a broader set of use cases than either of these applications, and offers an identical user experience across PC and iPad platforms. Digital signing software is specifically designed to sign/initial preconfigured documents. Digital Alternatives can be used to sign any document, without training or OCR software. Note taking applications are often only available for the tablet, designed for PDF annotation. Digital Alternatives works on PCs and iPads and offers the ability to sign, read, annotate, complete forms, synch across devices and share documents with others.

How does Digital Alternatives differ from other PDF viewer and annotation utilities?

Most of the leading pdf viewer and annotation utilities are not enterprise apps; rather they operate as stand-alone applications that the user downloads and installs on their own. Digital Alternatives is an enterprise solution, and includes the following capabilities that most other tools lack:

  • Enterprise, private cloud storage for improved security
  • Integration with Microsoft Active Directory for user authentication
  • Integration with Microsoft Exchange for seamless sharing
  • Centralized administration and deployment for enhanced management
  • Rich usage logging and analytics for improved insight

In addition, Digital Alternatives offers more user benefits than typical PDF viewers, including consistent experience on PCs and mobile devices; easy form completion for any type of common office document today, and real-time sync across devices.

What devices does the software support?

  • Microsoft Windows 7 and 8
  • Apple iPad 2/3/4 generation running iOS 7.1 up to 8.0.2
  • Not supported: Mac OS, Android, Blackberry, Windows Mobile
  • Android is on the roadmap, and the program is actively investigating expanded client support

How is the application installed on a user’s PC and/or tablet?

Xerox installs and manages the Digital Alternatives server application for IT. The IT department pushes the PC client to user desktops using a “Silent” or automatic install (similar to how print drivers are installed). The iPad App is user downloadable from the Apple Store. Users may also install Digital Alternatives on a second work computer or home PC with an available executable (.exe) package.

What is required of the customer when setting up Digital Alternatives?

A partial list of customer requirements follows below. For a complete list, please contact your Xerox representative.

  • Provisioning of server hardware, server software and server connectivity
  • Support to Xerox personnel in configuring and implementing the Digital Alternatives solution for on premise deployment
  • Ability to connect the on premise server to Microsoft Azure cloud server environment via proxy or DMZ to support user registration, solution licensing, and remote capabilities for end users
  • Ability to support server application connection to the domain
